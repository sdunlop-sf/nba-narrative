How It Works
From customer signal to
personalised action in milliseconds.
Every NBA follows the same five-step loop — running continuously, across 12 million customer profiles, in real time.
Step 01
Signal fires
Till transaction, Loyalty milestone, app session, browsing pattern, project completion, seasonal trigger, or inactivity threshold. Data Cloud segment membership updates in real time.
Step 02
Profile read
Full Living Profile fetched — purchase history, project phase, PowerPass status, Loyalty tier, seasonal context, suppression flags, consent status.
Step 03
NBAs scored
Einstein Recipes + Studio run propensity models against the NBA catalogue. Each action gets an eligibility check and a probability score. Ranked shortlist returned in milliseconds.
Step 04
Best action arbitrated
Business rules apply — frequency caps, Loyalty mission priority, channel preference, commercial margins. One winning NBA. Governed, consistent, auditable.
Step 05
Executed + fed back
In-app prompt, personalised email, agent brief, or push notification fires. Outcome is logged → Einstein retrains → next decision is smarter.
