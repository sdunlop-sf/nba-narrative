How It Works
Four steps from download
to a branded, tracked page.
No build tooling. No frameworks. Plain HTML + CSS that opens in any browser.
STEP 01
Download & unzip
One zip file, 8.4 MB. Drop the design-kit/ folder into your project.
STEP 02
Duplicate the template
Open template.html, save as your page name. Replace placeholder copy and swap the customer logo.
STEP 03
Set the accent colour
One line of CSS in the <head> — --accent: #yourHex. The whole page rebrands.
STEP 04
Deploy & share
Push to any static host — GitHub Pages, Netlify, Cloudflare. The gate tracks visitors automatically.
The single-line gate
<!-- Paste above </body> to lock the page + start tracking --> <script src="gate.js" data-page="Customer Narrative Name"></script>
