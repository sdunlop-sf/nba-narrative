Defining the Building Blocks · 10:45 – 11:30

Three groups. Three outputs.

Group 1
Systems of Record
Data & Signals
Map every system that holds a relevant customer signal. Identify where data lives, what triggers exist, and what gaps need bridging before the pilot.
  • ? Where does each customer signal live today?
  • ? Which of these are already in Salesforce?
  • ? What data is missing for the top 3 use cases?
Output: Systems Map
Group 2
Templates & Standards
Actions & Content
Define what each NBA looks like — trigger, message, channel, success metric. Build a reusable template the team can apply to every use case going forward.
  • ? What triggers each NBA — and how specific?
  • ? What does a 'good' message look like for Xero?
  • ? How do you define success for each use case?
Output: Templates & Standards Doc
Group 3
Channel Orchestration
Delivery & Execution
Map how NBAs reach the customer. Which channels are available, which teams are in scope, and where the human/agent handoff sits for each use case.
  • ? Which channels can you act on today vs. later?
  • ? What should an agent handle vs. a rep?
  • ? What are the contact frequency and suppression rules?
Output: Channel Orchestration Map

From Workshop to Execution · 12:15 – 12:30

The 4-week plan to pilot.

Week 1
Foundation
  • Lock pilot crew and schedule kickoff
  • Finalise top 3 use cases and success metrics
  • Deliver SIC brief for exec sign-off
  • Begin data audit against systems map
Week 2
Data & Signal
  • Connect core data sources to Data Cloud
  • Validate unified customer profiles
  • Define trigger conditions for each use case
  • Draft NBA templates — content and channels
Week 3
Build & Configure
  • Stand up Einstein scoring model
  • Configure NBA flows and adjudication logic
  • Build rep-facing surfaces in Salesforce
  • Test with synthetic data — sign off on logic
Week 4
Pilot Launch
  • Enable pilot crew and run first live actions
  • Monitor feedback loop — captures outcomes
  • Weekly review cadence begins
  • Prepare expansion brief for next phase

Every output from today feeds directly into Week 1. The plan starts when we leave this room.
